Book Cover

Tools: ChatGPT, Adobe Firefly, Adobe Photoshop

This project was accomplished with the use of firstly, ChatGPT to create a title, an author, and subtitle. I ended up using my own name as the author just because. I then created a cover image for the book using the text-image feature in Firefly. After that, I assembled everything together like a sandwhich, taking all the different texts and images to create the delicious final version of the book cover.
